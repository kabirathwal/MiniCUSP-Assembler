Project Name: MiniCUSP (MICA) Assembler

## Description:
A MiniCUSP (MICA) assembler built from scratch to simulate a real assembler and explore how they work

## Features:
- Runs with two command line arguments: input assembly code file (.csp), output file (.txt)
- Unlike a real assembler, this outputs into text/ASCII format rather than binary

## Technologies used:
- C++
